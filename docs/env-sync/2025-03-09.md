# Allineamento ambienti Vercel — 09/03/2025

Questa nota registra l'esecuzione del check "Allineamento ambienti" richiesto dalla checklist Beta.

## Procedura
1. Esegui `vercel env pull .env.vercel.preview --environment preview` e `vercel env pull .env.vercel.production --environment production` dalla root del progetto (i file restano ignorati da Git).
2. Assicurati che `.env.local` sia aggiornato con gli stessi valori usati nello sviluppo locale.
3. Lancia il comando:
   ```bash
   node scripts/check-vercel-env.mjs --local=.env.local --preview=.env.vercel.preview --production=.env.vercel.production
   ```
4. In caso di errori, aggiorna le variabili mancanti su Vercel e ripeti il comando finché Preview/Production risultano coerenti con `.env.local`.
